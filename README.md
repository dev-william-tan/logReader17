# Log Parsing Project 

Simple log parsing project in Java 17 

### Pre-requisite
* JDK 17 or later 
* Gradle

### Build Project
`./gradlew build`

### Run Project
`./gradlew run`

### Run Tests
`./gradlew test`

Assumptions:

Log files follow the Common Log Format(CLF)

### Dependencies
* Java 17 
* Gradle 
* Log4j2
