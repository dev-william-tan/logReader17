# Log Parsing Project 

Simple log parsing project in Java 17 
that reads and processes some fixed example log data. 

### Pre-requisite
* JDK 17 or later 
* Gradle

### Installation
`./gradlew clean build`

### Run Project
in Terminal run
`java -jar build/libs/logReader-app.jar `

### Run Tests
`./gradlew test`

Assumptions:
Example logs are in Apache Log format 

### Dependencies
* Java 17 
* Gradle 
* Log4j2
